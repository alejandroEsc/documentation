The `cluster-to-redpanda-migration` application is a single binary that provides a way to  migrate Cluster and Console deprecated Kubernetes resources to the new Redpanda resources. You can install the `cluster-to-redpanda-migration` binary as a standalone tool for self-hosted and cloud deployments, as well as Kubernetes deployments.


After you install `cluster-to-redpanda-migration`, you can use it to:

- Migrate Cluster and Console deprecated Kubernetes resources to the new Redpanda resources.

To install the `cluster-to-redpanda-migration` binary as a standalone application, follow the instructions for your operating system.
